## RMA Homepage Hero Media (Required Assets)

### Requirements
- **Format**: JPG, WebP preferred (JPG OK)
- **Minimum size**: **1920×1080**
- **Orientation**: Landscape
- **Style**: Professional documentary / corporate / institutional (no watermarks)
- **Licensing**: Safe for commercial use (keep license records)
- **Context**: Prefer Zambia/Africa mining + supplier development + workshops
- **Compression**: Optimized for web (avoid huge files in GitHub)
